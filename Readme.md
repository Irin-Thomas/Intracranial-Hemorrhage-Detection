# Intracranial Hemorrhage Detection using ML & Deep Learning

## Overview
This project focuses on detecting intracranial hemorrhage from non-contrast CT scan images using Machine Learning and Deep Learning approaches.

The project evaluates:
- Multi-Layer Perceptron (MLP)
- LightGBM
- CNN
- MobileNetV2
- EfficientNetB0
- ResNet50

Additional techniques:
- CLAHE preprocessing
- Transfer Learning
- Monte Carlo Dropout
- Uncertainty Quantification
- Model Calibration

## Dataset
CQ500 Dataset
